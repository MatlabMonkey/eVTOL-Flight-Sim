HW34 and HW56 Trace

What Each Live Script Does

HW34_sensors_rootlocus.mlx is the setup and analysis notebook. It shows the new wrapper architecture, checks the sensors against truth, shows one bias-on versus bias-off comparison, prints the open-loop eigenvalues, and displays the damper, PD, and PI-D root locus figures.

HW56_control_demos.mlx is the demo notebook. It runs one longitudinal closed-loop case and one lateral closed-loop case from the trimmed cruise setup and replots the main state and command histories.

Call Path

HW56_control_demos.mlx calls run_longitudinal_demo.m and run_lateral_demo.m. Those helpers run Brown_Control_Sim.slx, and that wrapper references Brown_6DOF_Plant.slx as the inner 6-DoF plant.

HW34_sensors_rootlocus.mlx calls run_sensor_checkout.m, linearize_cruise_plant.m, design_dampers.m, design_pd.m, and design_pid.m. The linearization path uses Brown_Control_Analysis_Helper.slx and the referenced plant.

What Was Added In The Last Controls Pass

  • Brown_Control_Sim.slx: new outer wrapper for sensors, estimator, controller, commands, and logging.
  • Brown_Control_Analysis_Helper.slx: scalarized analysis model for linmod().
  • Brown_6DOF_Plant.slx: kept as the reusable referenced plant.
  • scripts/simulink/build_control_wrapper_model.m: rebuilds the wrapper around the referenced plant.
  • scripts/simulink/build_control_analysis_helper.m: rebuilds the analysis helper used by the root-locus scripts.
  • scripts/control/setup_cruise_trim_case.m: loads the Homework 7 cruise trim result and uses it as the controller baseline.

Important Fixes During This Pass

  • The wrapper plant input order was corrected so the grouped collective and surface inputs go to the right plant ports.
  • The estimator path was updated so the complementary filter can use the attitude measurement as a stabilizing correction term.
  • The trim baseline was cleaned up so the motor-group trim vector is reserved for differential controller offsets instead of duplicating the grouped collective trim.
  • The wrapper collective wiring was corrected so the trimmed front and rear collective RPM values actually reach the plant during the demo runs.
